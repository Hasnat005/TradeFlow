# TradeFlow

Monorepo for the TradeFlow fintech platform.

## Repository Structure

```text
tradeflow/
├── frontend/
│   └── mobile-app/
├── backend/
│   └── api-server/
├── docs/
├── .gitignore
└── README.md
```

## Folders

- `frontend/mobile-app`: React Native mobile application (Expo + TypeScript)
- `backend/api-server`: Node.js API server
- `docs`: Architecture diagrams and project documentation
