# TradeFlow Mobile App

Production-ready React Native fintech starter built with Expo and TypeScript.

## Stack

- React Native + Expo
- TypeScript
- React Navigation
- Axios
- React Query
- Zustand
- ESLint + Prettier

## Project Structure

```text
src/
  components/
  screens/
  features/
  navigation/
  services/
  hooks/
  store/
  utils/
  theme/
```

## Environment Variables

1. Copy `.env.example` to `.env`
2. Update values as needed

Required:

- `EXPO_PUBLIC_API_BASE_URL`

## Scripts

- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`
- `npm run lint`
- `npm run typecheck`
- `npm run format`
- `npm run format:check`
